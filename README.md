# IPL Data Analysis Project 🏏

## 📌 Project Overview
This project analyzes IPL (Indian Premier League) dataset using Python.
The aim is to explore match statistics, team performance, and player insights.

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Key Analysis Performed
- Data Cleaning
- Team performance comparison
- Top players analysis
- Match result visualization

## 📂 Files in this Repository
- ipl_analysis.ipynb → Main analysis notebook
- Dataset file (if added)

## 🚀 Conclusion
This project helps understand IPL trends and performance insights using data analysis techniques.
